# Zombie Game

this is mfirst time useing pgame to make games, its a 2D game
where you shoot zombies, the higher the score more zombies
spawn but right now its not fully polished

## Controls

* W / A / S / D → Move
* Left Click → Shoot
* Space → Restart after Game Over

## things you need

* python 3.12
* pygame
* random
